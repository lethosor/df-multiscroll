## MultiScroll ##

This is a plugin for [Dwarf Fortress](http://bay12games.com) / [DFHack](http://github.com/dfhack/dfhack) that adds multitouch map scrolling on OS X.

Set VSYNC in DF init.txt to YES for better visual results.
